# Steganography LSB PNG

## What is it for?
You can use it for hiding message inside PNG or WAV file

## Requirements
- Stegano module
``` pip install stegano ```

- Python
``` sudo apt install python3 ```

## How to create venv
``` python3 -m venv *name_of_venv* ```

and then activate venv
``` source *name_of_venv*/bin/activate ```

## How to run
``` python3 code.py ```
